## 功能

- **后勤**：自动收取后勤，~~允许重置后勤对齐进度。~~
- **日常**：消耗动能、势能，完成每日事务，领取协议及活跃度奖励。
- **刷本**：仅支持部分副本拖尸、打捞（如8-1N），视情况适配活动打捞。
- **托管**：定时启动，托管设置项目。

## 常见问题

> **为什么费马不能帮我打开游戏？**<p>
MAAFramework下的win32状态，总是先捕获到一个窗口再开始运行，费马无法在没有打开游戏的情况下捕获游戏窗口╮(╯▽╰)╭</p>

> **为什么脚本会抢鼠标？**<p>
国服目前只有桌面端，**运行脚本时游戏窗口必须保持在前台**，所以本项目这点上不像Alas、MAA等项目，而是更像BetterGI、三月七，必然抢鼠标(ó﹏ò｡)</p>

> **为什么找不到自动后勤的功能？如何设置我的后勤？**<p>
后勤完成界面会于游戏停留在主页面时自动弹出，如果不及时点掉必然报错，故费马自带了**不可取消**的后勤重复功能，无需单独选择，而是**每个任务都会自动收后勤**。<p>
同时，MFA自带了定时启动功能，只需将定时启动的间隔修改为后勤跨度，保证后勤结束时费马会定时启动，就可以达到无缝后勤的效果。</p>

> **为什么脚本点击位置偏下 / 部署了错误的梯队？**<p>
这是GDI的通病，请按照如下流程操作<p>　1. 在费马中将触控模式**从GDI改为其它**<p>　2. 点击“开始任务”后立即点击“停止任务”<p>　3. 在费马中将触控模式**从其它改回GDI**<p>　　即可恢复正常</p>

> **为什么开始任务后游戏窗口发生撕裂？**<p>
这是YM特色，不得不品，可以通过**快速点击两次任务栏的游戏进程**（先最小化再放到前台）解决。
